cookies.txt-importer-for-chrome

Description

Fork of zhad3’s but instead of opening Options page it shows up a popup window, like so.

This extension lets you import cookies through one or more cookies.txt files which are saved in the netscape format. See The Unofficial Cookie FAQ for more information about the format.
